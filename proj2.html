ASSEMBLER

Project Mission:

Create a program that can convert assembly instructions to machine language

Output of the completed assembler- machine code




The assembler is a program that has been tested and verified with many different cases

What does it do?

The assembler takes in assembly instructions and converts the instruction to machine code which is just 1's and 0's! This project included parsing data input, creating functions that dealt with the different instructions, as well as functions that perform mathematical operations to keep the sign of the numbers and maintain its length
